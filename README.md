# Personal Portfolio

A simple personal portfolio website built with HTML, CSS, and JavaScript.

## Getting Started

1. Install dependencies:
   ```
   npm install
   ```

2. Start the development server:
   ```
   npm start
   ```

3. Open your browser and navigate to `http://localhost:8080` (or the port shown in the terminal).

## Project Structure

- `index.html`: Main HTML file
- `css/styles.css`: Stylesheet
- `js/script.js`: JavaScript for interactivity
- `images/`: Directory for images (add your photos here)
- `package.json`: Project configuration

## Customization

- Edit `index.html` to update content
- Modify `css/styles.css` for styling
- Add functionality in `js/script.js`
- Replace placeholder text and links with your own information

## Troubleshooting

- If `npm start` doesn't work, ensure Node.js and npm are installed.
- Check that port 8080 is not in use by another application.
